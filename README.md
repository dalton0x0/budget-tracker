# Budget Tracker

Application React de gestion de budget personnel, permettant de suivre ses revenus et dépenses.

## Fonctionnalités

- Ajouter des revenus et des dépenses
- Visualiser le solde total, les revenus et les dépenses
- Filtrer les transactions par type (tous, revenus, dépenses)
- Tri automatique par date décroissante
- Persistance des données via localStorage
- Validation des champs du formulaire

## Technologies

- React 19 avec Vite
- Hooks (`useState`, `useEffect`)
- Bootstrap 5 + Bootstrap Icons
- localStorage pour la persistance

## Installation

```bash
npm install
npm run dev
```
